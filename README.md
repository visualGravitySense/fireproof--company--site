# Fire Proof Company Website

React + Vite сайт для Fire Proof Company - противопожарные услуги для строительства и промышленности.

## Структура проекта

Проект создан на основе плана из `research.md` и включает:

- **Главная страница** - героический блок, статистика, услуги, социальные доказательства
- **О компании** - история, команда, сертификаты, партнеры
- **Услуги** - детальные страницы для каждой услуги
- **Проекты** - портфолио по категориям (коммерческие, промышленные, жилые)
- **Блог** - статьи и база знаний
- **Ресурсы** - руководства, чеклисты, FAQ, нормативы
- **Материалы** - информация о партнерских материалах
- **Контакты** - форма обратной связи и контактная информация

## Технологии

- React 19
- TypeScript
- Vite
- React Router DOM

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build

# Просмотр production сборки
npm run preview
```

## Роутинг

Все маршруты настроены в `App.tsx`:
- `/` - Главная
- `/about` - О компании
- `/about/:section` - Подразделы (history, team, certificates, partners)
- `/services` - Услуги
- `/services/:service` - Детальная страница услуги
- `/projects` - Проекты
- `/projects/:category` - Проекты по категориям
- `/blog` - Блог
- `/blog/:slug` - Статья блога
- `/resources` - Ресурсы
- `/materials` - Материалы
- `/materials/:material` - Детальная страница материала
- `/contact` - Контакты
